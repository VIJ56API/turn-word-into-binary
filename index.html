📊 How It Works (Workflow)

1. User enters text string.
2. JS loops through each character via charCodeAt().
3. Decimal ASCII value is converted to base-2 (Binary).
4. Result is padStart(8, '0') to ensure 8-bit format.
5. Reverse logic uses parseInt(bin, 2) to decode.
6. Final output is displayed in the result box.
7. type anywhere peace to active blue mode .
